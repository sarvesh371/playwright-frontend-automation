## Setup
1. Clone this project by doing:
```
$ git clone https://github.com/sarvesh371/playwright-frontend-automation.git
```
2. Go to the folder you've just cloned the code and execute:
```
$ npm install
$ npx playwright test
$ npx playwright show-report
```
